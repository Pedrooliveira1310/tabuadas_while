📘 Gerador de Tabuadas com While

Projeto desenvolvido para praticar a estrutura de repetição while em JavaScript, gerando automaticamente as tabuadas do 1 ao 10.

📌 Descrição

O Gerador de Tabuadas (While) é uma aplicação web simples que exibe as tabuadas do 1 ao 10, utilizando a estrutura de repetição while para calcular e exibir os resultados dinamicamente na tela.

O objetivo principal da atividade é reforçar o uso de:

Estrutura de repetição while

Manipulação do DOM

Organização de conteúdo com HTML

Estilização com CSS

🛠️ Tecnologias Utilizadas

HTML5 → Estrutura da página

CSS3 → Estilização e layout responsivo

JavaScript → Lógica de programação com while

🚀 Funcionalidades

✅ Botão para gerar as tabuadas

✅ Exibição das tabuadas do 1 ao 10

✅ Cálculo automático de 1x1 até 10x10

✅ Interface moderna com efeito visual em gradiente

✅ Organização das tabuadas em cards individuais

📚 Conceitos Trabalhados
🔁 Estrutura de Repetição while

A estrutura while executa um bloco de código enquanto uma condição for verdadeira.


No projeto, o while foi utilizado para:

Repetir os cálculos de multiplicação

Gerar dinamicamente cada linha da tabuada

Controlar o incremento dos números

🎯 Objetivo da Atividade

Praticar lógica de programação

Fixar o uso da estrutura while

Melhorar a manipulação do DOM

Desenvolver organização visual com CSS

📷 Interface

A aplicação apresenta:

Título centralizado

Botão para gerar as tabuadas

Cards organizados em grade

Fundo em gradiente vermelho escuro
